# [Bibliography](@id bibliography)

The complete, canonical bibliography for MacroEconometricModels.jl. Every work cited anywhere in the
documentation has exactly one entry here, listed alphabetically by first-author surname. Individual
method pages carry their own topic-scoped reference lists; this page is the master index.

For method-specific references at runtime, `refs(model)` and `refs(io, :method)` emit the relevant
entries in text, LaTeX, BibTeX, or HTML format (see [How to Cite](@ref citation)).
