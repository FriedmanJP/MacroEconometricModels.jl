Classical IO Analysis

Demand-driven (Leontief) and supply-driven (Ghosh) analysis, multipliers, linkages, key-sector identification, structural decomposition, and hypothetical extraction. All examples use the built-in load_example(:wiot) table.

using MacroEconometricModels
io = load_example(:wiot)

Leontief and Ghosh inverses

The technical-coefficients matrix is $A = Z\hat{x}^{-1}$ and the Leontief inverse (total requirements) is $L = (I-A)^{-1}$. The supply-side analogues are the allocation-coefficients matrix $B = \hat{x}^{-1}Z$ and the Ghosh inverse $G = (I-B)^{-1}$.

A = technical_coefficients(io)
L = leontief_inverse(io)
L
2×2 Matrix{Float64}:
 1.25413   0.330033
 0.264026  1.12211
m = leontief(io)          # bundles A, L, x
g = ghosh(io)             # bundles B, G, x
g.G
2×2 Matrix{Float64}:
 1.25413   0.660066
 0.132013  1.12211

The two inverses are linked by $G = \hat{x}^{-1} L \hat{x}$.

Multipliers

multipliers returns sectoral output, income, or employment multipliers, as Type I (open model) or Type II (closed with respect to households).

multipliers(io; kind=:output, type=:I)
     I output multipliers

  Sector          Multiplier 

  Agriculture         1.5182
  Manufacturing       1.4521


multipliers(io; kind=:income, type=:I)    # value-added multipliers
     I income multipliers

  Sector          Multiplier 

  Agriculture            1.0
  Manufacturing          1.0

Type II multipliers close the model with respect to households (using the compensation row as household income and final-demand shares as consumption), adding induced effects:

multipliers(io; kind=:output, type=:II)
    II output multipliers

  Sector          Multiplier 

  Agriculture         3.5512
  Manufacturing       4.0924

Linkages and key sectors

Backward linkages are column sums of L; forward linkages are row sums of the Ghosh inverse G (or of L with forward=:leontief). The Rasmussen power-of-dispersion ($U_i$) and sensitivity-of-dispersion ($U_j$) indices normalize these by their overall average, and the (U_i, U_j) quadrants give a key-sector classification.

lk = linkages(io)
         Linkages (Rasmussen dispersion indices)

  Sector          U_i (backward)   U_j (forward)   Class 

  Agriculture             1.0222          1.2083   key
  Manufacturing           0.9778          0.7917   weak


key_sectors(io)           # :key / :forward / :backward / :weak per sector
2-element Vector{Symbol}:
 :key
 :weak

Structural decomposition analysis

sda decomposes the change in gross output between two periods into a technology (ΔL) effect and a final-demand (Δy) effect. The additive two-polar decomposition is exact (zero residual); a multiplicative variant is also available.

io2 = IOData(io.Z .* 1.1, io.Y .* 1.2, [330.0 1100.0; 385.0 440.0];
             sectors=io.sectors, check=false)
r = sda(io, io2; method=:additive)
r.effects
Dict{Symbol, Vector{Float64}} with 2 entries:
  :L => [-59.5833, -27.5]
  :Y => [194.583, 397.5]

Hypothetical extraction

hypothetical_extraction measures the output loss from removing one or more sectors (by index or by name).

hypothetical_extraction(io, "Agriculture")
Hypothetical extraction of sector(s) [1]
  total output loss: 1210.5263

multipliers(io; kind=:output, type=:I) -> IOMultipliers

Sectoral multipliers. kind selects :output (column sums of L), :income (value-added weighted), or :employment (jobs-weighted, needs an employment extension). type=:II closes the model with respect to households.

source
MacroEconometricModels.linkages — Function
linkages(io; forward=:ghosh) -> LinkageResult

Backward linkages (column sums of L) and forward linkages (row sums of the Ghosh inverse G by default, or of L with forward=:leontief). Rasmussen power-of-dispersion (Ui) and sensitivity-of-dispersion (Uj) indices are the linkages normalized by their overall average, and the per-sector classification follows the (Ui, Uj) quadrants.

source

Per-sector key-sector classification (:key/:forward/:backward/:weak).

source
MacroEconometricModels.sda — Function
sda(io0, io1; method=:additive, factors=:LY, average=:two_polar) -> SDAResult

Decompose the change in gross output x = L y between two periods into a technology (ΔL) effect and a final-demand (Δy) effect.

  • method=:additive uses the Dietzenbacher & Los (1998) two-polar average, which is exact (zero residual).
  • method=:multiplicative returns the analogous ratio decomposition.

factors=:LY selects the two-factor L/Y decomposition (a stable hook for finer splits); average=:two_polar selects the averaging scheme.

source
MacroEconometricModels.hypothetical_extraction — Function
hypothetical_extraction(io, sectors) -> ExtractionResult

Total-output loss from removing sectors (an index, vector of indices, or sector name(s)): zero the extracted rows and columns of A and the extracted final demand, re-solve x = (I−A)⁻¹ y, and compare to the baseline.
